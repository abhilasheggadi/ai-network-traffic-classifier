# AI Network Traffic Classifier

Machine learning-based network traffic classification system. Build step-by-step.

## 📁 Folder Structure

```
├── backend/          # FastAPI API (Step 2)
├── frontend/         # Streamlit Dashboard (Step 3)
├── model/            # Model training (Step 1)
├── dataset/          # Data files
├── utils/            # Helpers
├── data/             # Outputs
└── requirements.txt
```

## 🎯 What We're Building

- **Input:** Network traffic features
- **Process:** ML classifier (Random Forest)
- **Output:** Traffic type prediction

## 📊 Traffic Classes

- **Normal** - Legitimate traffic
- **DoS** - Single source attack
- **DDoS** - Distributed attack
- **Port Scan** - Reconnaissance
- **Brute Force** - Password guessing
- **Botnet** - Infected device activity
